This add-in demonstrates how to use single sign-on by making a call to Microsoft Graph to get user profile data.

To use this add-in:

  • Click the Get My User Profile Information button.
  • If you are not signed into Office, you are prompted to sign in.
  • You may also be prompted to accept the app's permissions request.
  • Your user profile information will be displayed in the document.
