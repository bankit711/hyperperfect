Getting Started

After installing the HyperPerfect add-in, you can access it from the Home tab in Excel. Click on the "Show HyperPerfect" button to open the task pane.

The HyperPerfect interface consists of two main sections:

  • Clean - Tools for cleaning and standardizing your data
  • Analyze - Tools for analyzing and visualizing your data
Tip: You can navigate between these sections using the navigation menu at the top of the task pane.

Clean Data Feature

The Clean Data feature provides tools to prepare your data for analysis by cleaning and standardizing it.

Formula Clean

The Formula Clean feature helps you process your data using formulas, making it easier to work with complex datasets.

To use Formula Clean:

  1. Select your input sheet from the dropdown menu
  2. Configure the cleaning options as needed
  3. Click the "Clean Source Data with Formulas" button

Data Conversion Options

HyperPerfect offers several data conversion options to standardize your data:

  • Monthly to Annual Adjustment - Convert between monthly and annual revenue figures
  • Unit Conversion - Standardize units across your dataset
  • Adjust Partial Periods - Handle partial period amounts at the start or end of revenue streams
  • Plug Revenue Gaps - Fill gaps in revenue streams
  • Remove Overlapping Values - Identify and remove overlapping revenue entries
  • Exclude Small Revenue Amounts - Set a minimum threshold for revenue values

Analyze Data Feature

The Analyze Data feature provides tools to gain insights from your data through advanced analytics and visualizations.

Revenue Waterfall

The Revenue Waterfall feature helps you visualize changes in revenue over time, breaking down increases and decreases by various factors.

To generate a Revenue Waterfall:

  1. Select your source sheet from the dropdown menu
  2. Choose the metric you want to analyze
  3. Select the cohort detail level
  4. Apply any dimension filters if needed
  5. Click the "Revenue Waterfall" button

Pivot Tables

HyperPerfect can generate customized pivot tables to analyze your data from different perspectives.

To create Pivot Tables:

  1. Select your source sheet
  2. Choose the primary dimension
  3. Select the row configuration
  4. Choose the time period
  5. Click the "Run Pivot Tables" button

Frequently Asked Questions

General Questions

Q

What types of data can HyperPerfect process?

HyperPerfect is designed to work with various types of data, but it's particularly effective for financial and time-series data, especially revenue data with temporal dimensions.

Q

Does HyperPerfect modify my original data?

No, HyperPerfect creates new worksheets for processed data, keeping your original data intact.

Q

Can I customize the output formats?

Yes, HyperPerfect offers various customization options for both data cleaning and analysis outputs.

Q

Is my data secure when using HyperPerfect?

Yes, HyperPerfect processes all data locally within Excel and does not transmit your data to external servers.
